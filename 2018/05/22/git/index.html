Git基本操作

Git 是 linux之父Linus为了更好的管理他自己的开源项目linux系统而自己开发的一个软件，这个软件可以帮助Linux更好的进行开源开发。Git是目前世界上最先进的分布式版本控制系统，在很多的团队协作中起着很重要的作用。不仅仅应用于软件开发，在芯片的开发周期中，各种文件都是通过 git 版本控制系统进行管理。

基本概念

专用名词

  • Workspace：工作区
  • Index / Stage：暂存区
  • Repository：仓库区（或本地仓库）
  • Remote：远程仓库
